# 🌧️ Correlação Pluviométrica: Sistema SAI e Porto de Santos

Este projeto apresenta uma análise avançada de dados focada na correlação entre eventos climáticos extremos (precipitação na Serra do Mar) e a eficiência logística do escoamento rodoviário e portuário no Porto de Santos.

O dashboard foi desenvolvido como parte de uma pesquisa acadêmica na **FATEC Baixada Santista - Rubens Lara**, utilizando modelagem dimensional e arquitetura de dados moderna.
